# 🐍 Level 1: Python Foundations

Welcome to your first Python challenge! This repository contains 6 tasks designed to take you from "What is code?" to building your first interactive programs.

## 🛠 How to work:
1. Open the task file (e.g., `task_1_hello.py`).
2. Read the instructions inside the file.
3. Write your code and click **Run**.
4. Check the **Hints** below if you get stuck.

---

## 🏆 The Challenges

### ✅ Task 1: The Messenger
**Goal:** Make the computer speak.
- **Task:** Print a welcome message to Branksome Hall Asia.
- **What to expect:** You should see your text appear in the console.
- **Hint:** Computers need text to be inside "quotation marks."

### ✅ Task 2: The Treasure Box (Variables)
**Goal:** Store information for later.
- **Task:** Create a variable called `playerName` and assign your name to it. Print it.
- **Hint:** Think of a variable like a box with a label on the outside.

### ✅ Task 3: The Calculator
**Goal:** Basic Math.
- **Task:** Create two variables, `num1` and `num2`. Print the result of them added together.
- **Expectation:** If num1 is 10 and num2 is 5, it should show `15`.

### ✅ Task 4: The Chatbot
**Goal:** Getting user input.
- **Task:** Ask the user "What is your favorite color?" and then print "I like [color] too!"
- **Hint:** Use the `input()` function.

### ✅ Task 5: The Bouncer (Logic)
**Goal:** Making decisions with `if`.
- **Task:** Create a variable `age`. If age is 12 or more, print "Welcome to Grade 7!". Otherwise, print "Not quite yet!"

### ✅ Task 6: The Repeat Machine (Loops)
**Goal:** Do things many times.
- **Task:** Use a `while` loop to print the numbers 1 to 10.
